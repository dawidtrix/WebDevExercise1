VFX: Cherry blossom tree

Description:

This VFX is representing the crown/s of cherry blossom tree.
In this video there are 3 VFXs, each on a different branch with different size and amount of particles.
Additional effect supplementing this VFX is an effect of falling leaves.
However, apart from the cherry blossom effect itself there is also a VFX for grass.
It uses simple noise patern to locate particles and particle strips to visualize them as individual blades of grass.

VFX: Soul Engine

Description:

This VFX has been born from the boredom.
The main point was to create multiple "entities" supposievly moving at random directions but at the same time not leaving the scene.
This effect was acomplished thanks to the derivative function of a commonly used Perlin Noise.
Apart from flying entites, a small particles in the background as well as lightning flare can be notice in the background.
